# YouTube 광고 스킵 (Chrome 확장)

YouTube 영상의 인비디오 광고를 자동으로 넘기고, 재생 속도를 YouTube 기본 2배 제한 없이 조절하는 Manifest V3 확장입니다.

- **건너뛰기 광고**: "건너뛰기" 버튼을 자동으로 클릭 (광고가 끝으로 간 뒤 남는 "넘어가기" 버튼까지)
- **건너뛰기 불가 광고**: 음소거 + 영상 끝으로 점프시켜 순식간에 스킵
- **배너/오버레이 광고**: 닫기 버튼 자동 클릭
- **재생 속도 조절**: YouTube 2배 제한을 넘어 최대 5배까지(슬라이더/프리셋), 광고·영상 전환 후에도 유지
- **팝업 UI**: on/off 토글 + 재생 속도 컨트롤 + 지금까지 건너뛴 광고 카운터

## 설치 (개발자 모드 로드)

1. Chrome에서 `chrome://extensions` 접속
2. 우측 상단 **개발자 모드** 켜기
3. **압축해제된 확장 프로그램을 로드합니다** 클릭
4. 이 폴더(`youtube-ad-block`) 선택
5. YouTube를 열고 영상 재생 → 툴바 아이콘으로 동작/카운터 확인

> 코드를 수정하면 `chrome://extensions`에서 이 확장의 **새로고침** 버튼을 눌러 다시 로드하세요.

## 구성

| 파일 | 역할 |
|------|------|
| `manifest.json` | 확장 메타데이터 / 권한 / content script 등록 |
| `content.js` | YouTube 페이지에서 광고 감지·스킵 및 재생 속도 적용 |
| `popup.html` / `popup.js` | 툴바 팝업 UI (토글 + 재생 속도 + 카운터) |
| `icons/` | 확장 아이콘 |

## 동작 원리

YouTube는 광고를 영상 본 스트림과 같은 플레이어로 내보내기 때문에 네트워크 차단이 까다롭습니다.
그래서 이 확장은 **DOM 조작 방식**을 씁니다 — 플레이어에 `.ad-showing` 클래스가 붙는 순간을
감지해 건너뛰기 버튼을 클릭하거나, 불가능하면 영상을 끝으로 점프시킵니다.

## 한계 & 주의

- YouTube가 플레이어 마크업/클래스명을 바꾸면 셀렉터를 업데이트해야 합니다 (`content.js`의 `SKIP_BUTTON_SELECTORS`).
- 광고 차단은 YouTube 이용약관상 권장되지 않으며, 탐지 시 재생이 제한될 수 있습니다. **개인 학습/사용 용도**로만 사용하세요. 스토어 배포는 정책 위반일 수 있습니다.
- 더 강력한 차단을 원하면 [uBlock Origin](https://github.com/gorhill/uBlock) 같은 성숙한 필터 기반 확장도 참고하세요.
