YouTube 광고는 스킵,
속도는 2배 그 이상.

건너뛰기 광고를 자동으로 넘기고, 건너뛰기 불가 광고는 순식간에 지나칩니다. 재생 속도는 YouTube의 2배 제한을 넘어 최대 5배까지.

⬇ 다운로드 .zip · 무료 설치 방법 보기 ↓

기능

⏭️

건너뛰기 광고 자동 스킵

"건너뛰기" 버튼이 뜨면 즉시 클릭. 광고가 끝으로 간 뒤 남는 "넘어가기" 버튼까지 눌러줍니다.

⚡

건너뛰기 불가 광고 고속 처리

건너뛸 수 없는 광고는 음소거하고 영상 끝으로 점프시켜 순식간에 지나칩니다.

🚫

배너 · 오버레이 광고 닫기

영상 위에 뜨는 배너/오버레이 광고의 닫기 버튼을 자동으로 클릭합니다.

🏃

재생 속도 최대 5배

YouTube 기본 2배 제한을 넘어 최대 5배. 광고가 지나가거나 다음 영상으로 넘어가도 속도가 유지됩니다.

다운로드 & 설치

1

다운로드 후 압축 해제

위 다운로드 버튼(또는 아래 GitHub Releases)에서 youtube-ad-blocker.zip을 받아 더블클릭해 압축을 풉니다.

2

개발자 모드 켜기 — Developer mode

Chrome 주소창에 chrome://extensions를 입력해 들어간 뒤, 우측 상단의 개발자 모드 스위치를 켭니다.

3

압축해제된 확장 프로그램을 로드 — Load unpacked

왼쪽 위에 나타난 압축해제된 확장 프로그램을 로드합니다 버튼을 누르고, 1단계에서 압축을 푼 youtube-ad-blocker 폴더를 선택하면 설치 완료입니다.

개인 학습/사용 목적의 도구입니다. YouTube가 플레이어 마크업을 바꾸면 일시적으로 동작하지 않을 수 있으며, 광고 차단은 YouTube 이용약관상 권장되지 않습니다.
